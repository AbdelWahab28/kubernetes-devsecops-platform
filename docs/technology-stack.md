# Choix technologiques

| Domaine | Technologie | Justification |
|-|-|-|
| Virtualisation | Proxmox VE | Simulation environnement entreprise |
| Conteneurisation | Docker | Standard de packaging applicatif |
| Orchestration | Kubernetes | Gestion des applications distribuées |
| Cluster | kubeadm | Déploiement Kubernetes personnalisé |
| CI/CD | GitLab CI/CD | Automatisation DevOps |
| Sécurité | Trivy, Semgrep | Intégration sécurité |
| Monitoring | Prometheus/Grafana | Observabilité |
| Logs | Loki | Centralisation logs |
