# Besoins du projet


## Besoins fonctionnels

La plateforme doit permettre :

- Déployer une application Full Stack
- Automatiser les déploiements
- Gérer plusieurs environnements
- Superviser les applications


## Besoins techniques

La plateforme doit intégrer :

- Une infrastructure virtualisée
- Un cluster Kubernetes
- Une chaîne CI/CD
- Des outils de sécurité
- Une solution d'observabilité
