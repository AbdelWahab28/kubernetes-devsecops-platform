# Contexte du projet

Dans le cadre de sa stratégie de modernisation de l'enseignement des technologies Cloud Native, la mise en place d'une plateforme pédagogique permettant aux étudiants d'apprendre à déployer des applications web selon les pratiques professionnelles actuellement utilisées en entreprise deviens une necessité.

Cette plateforme s'appuie sur un cluster Kubernetes afin d'offrir un environnement de déploiement moderne intégrant les principes du DevOps et du DevSecOps. Elle doit permettre aux étudiants de manipuler les différentes étapes du cycle de vie d'une application, depuis sa conteneurisation jusqu'à son déploiement, sa supervision et sa maintenance.

## Problématique

Les technologies Cloud Native, Kubernetes et DevSecOps sont devenues essentielles dans les infrastructures modernes.

Cependant, leur apprentissage nécessite des environnements pratiques permettant de comprendre les problématiques réelles de déploiement, d'automatisation et de sécurité.

## Besoin identifié

Mettre en place une plateforme permettant aux étudiants de manipuler :

- Docker
- Kubernetes
- CI/CD
- Sécurité applicative
- Observabilité

dans un environnement proche de celui rencontré en entreprise.
